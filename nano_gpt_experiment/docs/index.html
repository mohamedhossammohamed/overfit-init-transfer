Independent research — pilot study

Does overfitting leave a reusable trace?

An experiment on initialization geometry in small transformers: if a network is trained to extreme overfitting on one corpus, does starting subsequent training from that state help, hurt, or make no difference at all?

Status: pilot trial in progress · results published as they are collected

The question

Training a network to convergence changes its weights in ways that are not well understood outside of the task it converged on. It is an open question whether that state is generically easier to train further, from having settled into some reusable structure, or generically harder, from having settled into a sharp minimum the optimizer struggles to leave.

This experiment tests that directly, using two text corpora that share almost no characters, so any effect observed cannot be explained by shared vocabulary or by the content itself transferring. What is being measured is closer to the shape of the loss landscape at a given starting point than to transfer learning in the usual sense.

Design

Path A
Random initialization, trained directly on the target corpus. The baseline.
Path B
Random initialization, overfit on a source corpus, then trained on the target corpus from that state — same step budget as Path A.
Path C
Random initialization, converged on synthetic random-character data, then trained on the target corpus — isolates whether the source corpus mattered specifically, or any prior convergence produces the same effect.
Depth
Two checkpoint depths are taken deterministically at 50% and 90% of the fixed token budget, and each becomes an independent starting point. Training uses a fixed token budget of 200 tokens per parameter, 10x the Chinchilla compute-optimal ratio, deliberately overshooting to induce memorization.

Scope

This is a small, character-level pilot — models between 100k and 2M parameters, two random seeds per configuration, run on personal hardware. It is exploratory, not confirmatory: a pattern here is a reason to look further, not a finding on its own. Every metric and checkpoint is published as collected, including null or negative results.
